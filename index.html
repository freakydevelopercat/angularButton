Inspinia inspired buttons

Needed files for functioning of switch buttons:
1.scripts/button.js
2.dist/switch/switch.css
3.angular script
Instantiate the buttonswitch directive inside of your controller. The button is infact a checkbox. The state of button is determined by checked attrbute. It can be empty string or checked.

<div buttonswitch color="inspiniayellow" size="medium"></div>
<div buttonswitch color="inspiniared" size="medium"></div>
<div buttonswitch color="inspiniablue" size="medium"></div>
<div buttonswitch color="inspiniagreen" size="medium"></div>

Switchery buttons

The switchery button is taken from site Switchery but it is modified so you can use it by inserting two documents (inside dist/switchery folder) and by inserting a piese of code inside your controller. This code can be also found inside button.js script.

angular.element(document).ready(function (){ var elem1 = document.querySelector('.js-switch'); var switchery = new Switchery(elem1, { color: '#7c8bc7' }) });

This solution only excludes an original jquery solution but keeps all functions from Switchery untouched.

var elem0 = document.querySelector('.js-switch'); var switchery = new Switchery(elem0, { color: '#7c8bc7' });
var elem1 = document.querySelector('.js-switch1'); var switchery = new Switchery(elem1, { color : '#64bd63', secondaryColor:'white', jackColor: '#64bd63', size:'small' });
var elem2 = document.querySelector('.js-switch2'); var switchery = new Switchery(elem2, { color : '#f8ac59', secondaryColor:'#f8ac59', jackColor: 'white', size:'large' });
